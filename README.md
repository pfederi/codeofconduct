# Pumpfoilers Code of Conduct - Switzerland

Eine Website für den Pumpfoilers Code of Conduct in der Schweiz mit modernem Design und Animationen.

## Features

- Responsive Design basierend auf Swiss Design (Müller Brockmann Grid)
- Maritimes Farbschema
- Animationen mit GSAP
- Unterstützerliste
- Unterzeichnerfunktion mit lokaler Speicherung
- Vollständiger Code of Conduct Text

## Installation und Entwicklung

### Voraussetzungen

- Node.js (Version 16 oder höher)
- npm oder yarn

### Installation

1. Repository klonen oder Dateien herunterladen
2. Abhängigkeiten installieren:

```bash
npm install
```

### Entwicklungsserver starten

```bash
npm run dev
```

Die Website ist dann unter http://localhost:3000 verfügbar.

### Für Produktion bauen

```bash
npm run build
```

Die Ausgabedateien befinden sich im `dist`-Verzeichnis.
